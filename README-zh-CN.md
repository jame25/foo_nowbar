## 功能特性

### 核心功能
- **双界面支持**：在默认界面 (DUI) 和分栏界面 (CUI) 中均可无缝使用。
- **专辑封面显示**：显示专辑封面，支持直角/圆角切换，可添加边距。
- **在线封面获取**：当本地/内嵌封面不可用时，通过 [foo_artwork](https://github.com/jame25/foo_artwork) 从在线来源获取封面。
- **曲目信息显示**：使用可自定义的标题格式化语法显示曲目信息（默认显示：标题和艺术家）。
- **DPI 感知**：在高 DPI 显示屏上能正确缩放，并自适应大小。

### 播放控制
- **播放/暂停**：中央播放按钮，用于切换播放状态。
- **停止**：可选的停止按钮（默认隐藏）- 停止播放。
- **当前音轨结束后停止**：可选的切换按钮（默认隐藏）- 在当前曲目播放完后停止；激活时显示强调色。
- **上一首/下一首**：在曲目间导航。
- **乱序播放**：切换乱序播放模式，带视觉指示器（激活时显示强调色）。
- **重复播放**：循环切换重复模式：关闭 → 全部 → 单曲（激活时显示强调色）。
- **进度条**：用于在曲目内跳转的交互式进度条。
  - 可选择胶囊形或矩形样式。
  - 显示已播放和剩余时间。
  - 鼠标悬停时显示光标位置的时间戳提示。

### 音量控制
- **音量滑块**：点击或拖动以调节音量（感知响度映射，与 foobar2000 默认曲线一致）。
- **静音切换**：点击音量图标可静音/取消静音。
- **鼠标滚轮支持**：在面板上滚动可调节音量。
- **分贝提示**：鼠标滚轮滚动时显示当前音量水平的提示。

### 特殊按钮
- **心形喜爱按钮**：点击标记喜爱当前曲目。
  - 可配置的标签：`%FEEDBACK%`、`%2003_LOVED%`、`%LFM_LOVED%`、`%SMP_LOVED%` 或 `%MOOD%`。
  - 标记喜爱时为红色，为空时为灰色。
  - 可通过设置隐藏。
- **迷你播放器启动**：快速启动迷你播放器的按钮。
  - 需要 [foo_traycontrols](https://github.com/jame25/foo_traycontrols) 插件。
  - 激活时为强调色，默认隐藏。
- **功能聚合按钮**：包含自动播放列表查询和播放选项的快捷菜单。
  - 自动播放列表查询：从未播放、近5天播放、未评级、评级为 3-5/4/5、我的收藏（标记喜爱的曲目）、近2周添加、相同艺术家/标题。
  - **无限播放**：切换以在播放列表结束时自动添加相似曲目（基于媒体库的流派匹配）。
  - **播放预览**：子菜单，可设置在每首曲目的部分播放后自动跳过（关闭 / 35% / 50% / 60秒）。
  - **设置**：快速链接以打开 Now Bar(播放控制面板) 设置页面。
- **最多 12 个自定义按钮**：完全可配置的操作按钮。
  - **按钮 1-6**：显示在面板上，通过设置页面配置。
  - **按钮 7-12**：隐藏，仅限键盘快捷键，通过配置文件配置。
  - 可执行的操作：打开 URL、[运行可执行文件](#highlight-word1)、打开文件夹或执行 foobar2000 菜单命令（[主菜单或上下文菜单](#highlight-word2)）。
  - **3D 按钮样式**：为按钮添加立体凸起效果，带渐变和阴影，按下时有触感动画，默认启用。
  - **状态反馈**：当命令处于选中/激活状态时（例如切换命令），Foobar2k 操作按钮会显示强调色。
  - 支持每个按钮自定义 Unicode 符号（可直接[输入字符或使用 U+XXXX 格式](#highlight-word3)）。
  - 每个按钮可自定义悬停提示文本。
  - 自适应布局：高度较大时为双行（按钮 1-3, 4-6），高度较小时为单行。
  - 播放期间可自动隐藏（可选，带 300ms 平滑淡入淡出动画）。
  - URL/可执行文件操作使用所选曲目（而非正在播放的曲目）进行标题格式化。
  - 配置档案：可保存/加载按钮设置（新建 / 重命名 / 删除 / 导出 / 导入）。
  - 配置文件位置：`<foobar2000_profile>\foo_nowbar_data\custom_buttons.conf`

### 可视化模式
- **频谱**：叠加于播放按钮下方的全宽频谱分析器。
  - 可配置条形宽度（细 / 正常 / 宽）、高度（低：面板高度的20% / 正常：33% / 高：50%）和样式（条形 / 曲线 / 多米诺）。
  - 细进度条位于顶部边缘，取代原来的进度条；悬停时放大以供跳转。
  - 时间显示移至右上角。
  - 默认 30 FPS，可选 60 FPS 模式。
- **波形进度条**：SoundCloud 风格的预计算波形，取代原来的进度条。
  - 基于 RMS 计算，区分已播放/未播放颜色。
  - 波形数据跨会话缓存到磁盘（`wavecache.db`）。
  - 支持完整的跳转功能，悬停时显示时间提示。

### 主题与外观
- **主题模式**：
  - **自动**：自动跟随 foobar2000 的主题。
  - **深色**：强制使用深色模式。
  - **浅色**：强制使用浅色模式。
  - **自定义**：与 DUI 或 CUI 配色方案同步。
- **背景样式**：
  - **纯色**：标准的纯色背景。
  - **专辑封面颜色**：从专辑封面的主色中提取的动态渐变。
  - **模糊专辑封面**：将专辑封面模糊作为背景。
- **进度/音量条样式**：胶囊形或矩形。
- **进度条显示**：支持显示或完全隐藏进度条/波形图/频谱。
- **进度条长度模式**：
  - **固定模式**（默认）：进度条从左侧喜爱按钮延伸到右侧功能聚合按钮，长度固定不变。
  - **缩放模式**：进度条随面板宽度动态调整，向两侧空白区域延伸（占用两侧可用空隙的70%）。
- **音量控制显示**：支持显示或完全隐藏音量图标及音量条。
- **播放图标样式**：样式1（可切换正常风格：强调色圆圈，深色图标或反相风格：强调色圆圈，白色图标） / 样式 2 （矩形轮廓、无强调色圆圈） / 样式3（实心矩形、无强调色圆圈）。
- **悬停圆形背景**：按钮上可选的悬停效果。
- **悬停放大效果**：播放控制按钮在悬停时放大 15%。
- **自定义字体**：可为左侧信息区域第一、二、三行及时间显示选择自定义字体（支持颜色、下划线等格式）。
- **10 种自定义颜色**：按钮强调色、播放/暂停强调色、进度条强调色、音量条强调色、按钮悬停颜色、频谱颜色、波形已播放/未播放颜色、进度条轨道色、音量条轨道色（每种颜色均可独立切换使用自定义颜色或主题派生颜色）。
- **平滑动画**：可切换的动画过渡效果（默认关闭以提升性能）。
  - 启用时：自定义按钮自动隐藏淡入淡出、背景切换淡入淡出。
  - 禁用时：即时切换，性能更佳。
- **专辑封面**：可切换专辑封面的可见性；可为专辑封面设置可选的边距。
- **在线封面**：当本地封面不可用时，从在线来源获取封面（需安装 [foo_artwork](https://github.com/jame25/foo_artwork) 插件）。

### 播放特性
- **跳过低等级曲目**：播放时自动跳过低等级的曲目（可配置阈值 1-3 星；需安装 foo_playcount 插件）。连续跳过 10 首曲目后暂停该功能，强制播放下一首，防止卡死。
- **无限播放**：当播放列表结束时，自动从媒体库中添加流派匹配的曲目（一次添加 15 首）。
- **播放预览**：快速浏览时，在每首曲目的部分播放后自动跳过（关闭 / 35% / 50% / 60秒）。

## 安装方法

1. **安装版 foobar2000**（已关联文件格式）：双击 `foo_nowbar*.fb2k-component` 文件安装<br>
   **便携版 foobar2000**：参数选项 → 组件 → 点击“安装” → 选择 `foo_nowbar*.fb2k-component` 文件 → 确定<br>
   
   **Releases 页面附件说明：**<br>
   - `foo_nowbar*.fb2k-component`：英文版<br>
   - `foo_nowbar-zh-CN*.fb2k-component`：中文版
   
2. 重启 foobar2000。
3. 添加面板：
   - **默认界面 (DUI)**：视图 → 布局 → 启用布局编辑模式 → 添加面板 → "播放控制面板"。
   - **分栏界面 (CUI)**：在您的布局中添加 "播放控制面板"。

### 自定义按钮选项卡

#### 配置档案
- **配置名称**：在已保存的按钮配置之间选择。
- **[...] 菜单**：新建、重命名、删除、导出、导入配置档案的选项。

#### 按钮配置
配置最多 6 个可见的自定义按钮（按钮 7-12 仅可通过配置文件配置）：
- **无**：按钮禁用。
- **打开链接**：在默认浏览器中打开一个 URL（支持标题格式化，使用所选曲目）。
- <a name="highlight-word1"></a>**运行程序**：启动一个外部程序（支持标题格式化，使用所选曲目，例如：路径输入 `D:\foobar2000\Spek\spek.exe`）。
- <a name="highlight-word2"></a>**Foobar2k 操作**：执行 foobar2000 菜单命令（例如：路径输入 `视图/ESLyric/桌面歌词/显示` 或上下文菜单 `文件操作/删除文件`）；当命令处于激活状态时显示强调色。
- **打开所在文件夹**：打开当前所选曲目所在的文件夹。

每个按钮还支持：
- <a name="highlight-word3"></a>**自定义图标**：支持 Unicode 符号，可直接输入字符或使用 U+XXXX 格式，比如字体选择 `Segoe UI`，字符处直接粘贴 `🎶`，或字体选择 `Segoe Fluent Icons`，字符处输入 `U+E8D6`。
- **提示文本**：悬停时显示的自定义文本（默认显示为 “按钮 #N”）。
- 当自定义图标缺失时，自动回退显示带数字的方形图标。

### 字体和颜色选项卡

#### 字体
- **左侧第一、二、三行**：左侧信息区域字体（支持颜色、下划线等格式）。
- **时间**：已播放/剩余时间的字体。

#### 颜色
每种颜色都有一个复选框：未勾选时使用来自 DUI/CUI 的主题颜色，勾选时使用自定义颜色。

| 颜色 | 默认（主题） | 描述 |
|-------|----------------|-------------|
| 按钮强调色 | DUI 高亮 / CUI 高亮 | 随机、重复等按钮的激活状态 |
| 播放/暂停按钮强调色 | DUI 高亮 / CUI 高亮 | 播放按钮的背景圆圈 |
| 进度条强调色 | DUI 高亮 / CUI 高亮 | 进度条填充色 |
| 进度条轨道色 | 主题派生 | 进度条未填充部分 |
| 按钮悬停色 | DUI 选定 / CUI 选定 | 按钮悬停圆圈 |
| 频谱颜色 | 自适应（跟随主题高亮色） / 纯色（可自定义） / 渐变（可自定义） / 频率（内置） |  |
| 波形强调色 | DUI 高亮 / CUI 高亮 | 波形中已播放的部分 |
| 波形轨道色 | DUI 文字 / CUI 文本 | 波形中未播放的部分 |
| 音量条强调色 | DUI 高亮 / CUI 高亮 | 音量条填充色 |
| 音量条轨道色 | 主题派生 | 音量条未填充部分 |

### 键盘快捷键
所有 12 个自定义按钮的操作都可以通过键盘快捷键触发：

1. 转到 **参数设置 → 快捷键**。
2. 在 **文件 → 播放控制面板** 下查找命令：
   - `自定义按钮 1` 至 `自定义按钮 12`
3. 为相应按钮的操作分配任意键盘快捷键。
